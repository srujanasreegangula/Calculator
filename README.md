# Calculator
# 🧮 Calculator Web Application

A simple, responsive, and user-friendly Calculator Web Application built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations and provides an intuitive interface for quick calculations while demonstrating fundamental front-end development concepts.

## 🌐 Live Demo

🚀 **Live Website:**
[Calculator Web Application](https://srujanasreegangula.github.io/Calculator/?utm_source=chatgpt.com)

## ✨ Features

* Basic arithmetic operations (+, −, ×, ÷)
* Clean and responsive user interface
* Real-time calculation display
* Clear and delete functionality
* Mobile-friendly design
* Fast and lightweight performance

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
